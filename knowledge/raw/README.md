# knowledge/raw/

Captured artifacts as-is: PDFs, transcripts, web archives, exports. Written by
`/wiki-ingest` (human-gated); organized by provenance (`papers/`, `web/`,
`notion/`, `slack/`, `repos/`, `talks/`) — subdirs appear as sources arrive.

Gitignored except this README — raw is a local cache, regenerable from the
`source_url`/`source_local` frontmatter in `../source/` pages.
